Grado PH is a weighted-grade calculator and class record system built around DepEd Order No. 015, s. 2026 — available on Android and Windows.

Android

Grado PH

An offline class record and grade computation app for K-12 teachers, phone-friendly and built for classrooms without reliable internet access.

  • Offline-first — all data stays on the device
  • Written Work / Performance Task / Examination weighted grading
  • Official DepEd transmutation table and letter grades (K-3)
  • Male/Female grouped class records, matching DepEd's paper format
  • PDF and CSV export of class records and final grades
  • Cross-term Final Grade averaging

Windows Desktop

Grado PH Desktop

The same DepEd grading system as a standalone Windows app, for teachers who prefer working on a laptop or computer at school or at home. No installation, no internet required.

  • Identical grading logic and DepEd-compliant PDF class records
  • Larger, easier-to-read grid for entering scores at a keyboard
  • In-app PDF preview before printing or saving
  • Portable — runs from a single folder, no install required
